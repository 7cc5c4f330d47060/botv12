# botv12 web panel

This is a web panel for botv12, a Minecraft bot originally designed for [Kaboom](https://kaboom.pw/) and its clones.

## How to install?

This is currently a non-functional mockup of what it will look like. The functional version will be based on the same codebase however, and will come bundled with the bot (in Git as a submodule).
