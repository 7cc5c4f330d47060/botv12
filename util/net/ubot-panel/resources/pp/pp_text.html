What is Collected

We do not collect much data on our users. We only will collect information necessary to run such as:

username, nickname, UUID, the game server you connect from, and command content to allow commands to work in game
username and password when logging in to the web panel
any files downloaded through the music command, to allow the music function to work correctly

This information is only stored in RAM and is erased when the process restarts. Username and password information is stored in a file on the disk.
